# SystrixOS

A lightweight OS built in C by Shadow.

---

## Commands

### Files

- `ls` list files
- `cat <file>` read a file
- `write <file> <text>` write to a file
- `append <file> <text>` append to a file
- `mktxt <f1> <f2>` create text files
- `touch <file>` create empty file
- `rm <file>` delete a file
- `cp <src> <dst>` copy a file
- `rename <old> <new>` rename a file

### System

- `uname` OS info
- `meminfo` memory usage
- `uptime` system uptime
- `ps` running processes
- `clear` clear screen
- `reboot` restart
- `halt` shutdown

### Programs

- `elf <file>` run a binary
- `photo <file>` view a PNG

---

## Tips

> Use UPPERCASE names for files on FAT32.

> Scroll up with Page Up, down with Page Down.

**Have fun building on SystrixOS!**
